## 👥 User Types

SuperBazi provides **tailored experiences** based on user type:

### 1. Regular User
- **Purpose:** Personal insight and self-discovery
- **Access Includes:**
  - BaZi chart calculation
  - Day Master and basic structure explanation
  - Daily personalized report (via WhatsApp, email, Telegram)
  - Limited saved profiles (depends on subscription)

### 2. Practitioner
- **Purpose:** Professional consultants and BaZi learners
- **Access Includes:**
  - Full BaZi calculations (10 Profiles, 5 Structures, Symbolic Stars, etc.)
  - Detailed life stage, luck pillar, and hidden stem analysis
  - Store and manage multiple client profiles
  - Generate professional reports (daily, weekly, monthly)

### 3. Corporate / Organization
- **Purpose:** Metaphysical insight for **HR, team dynamics, and strategic decisions**
- **Access Includes:**
  - Employee potential and compatibility analysis
  - Recruitment and team placement strategy
  - Bulk profile analysis and reporting
  - Optional API or dashboard integration

## 🎨 Branding & Tech Stack

**Brand Name:** SuperBazi

### **Tech Stack Overview**

#### **1. Web Application (Nuxt 3)**  
- **Nuxt 3 + TypeScript** — Modern front-end framework  
- **Vuetify 3** — UI components and layout system  
- **Pinia** — Global state management  
- **Axios + Interceptor** — API calls with error handling  
- **SCSS** — Styling with modular structure  
- **PM2** — Node.js production deployment  

#### **2. REST API (Node.js)**  
- **Node.js + Express** — Backend framework for RESTful services  
- **PostgreSQL** — Primary relational database  
- **Redis** — Caching and session management  
- **Sequelize** — ORM for PostgreSQL  
- **JSON Web Token (JWT)** — Authentication & authorization  
- **lunar-javascript** — BaZi, lunar, and Chinese metaphysics calculations  
- **Socket.IO** — Real-time events & notifications  
- **Winston** — Logging and monitoring  

#### **3. Mobile Application**  
- **Flutter** — Cross-platform mobile app development (iOS & Android)
